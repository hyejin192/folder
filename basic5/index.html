table 만들기(표만들기)

html에서 표를 구현할 때

  • tr(table row):행
  • th(table heading):행,열의 제목(굵은체,가운데 정렬 적용됨)
  • td(table data):열(행의 내용)
  • 속성 colspan="2":2칸의 열을 병합
  • 속성 rowspan="2":2칸의 행을 병합
